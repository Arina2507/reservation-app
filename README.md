# Reservation App

## Description
Reservation system developed using TDD and DevOps principles.

## Features
- Create reservation
- Prevent duplicate reservations
- Validate input data

## Technologies
- Node.js
- Express
- Jest
- Docker
- GitHub Actions
- Kubernetes

## Run locally
npm install
npm test

## CI/CD
Pipeline runs tests automatically on push.

## Testing
- Unit tests
- Integration tests
- Coverage enabled (~80%)
